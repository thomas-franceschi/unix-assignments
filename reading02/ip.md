TLDR - ip
==========

Overview
--------

[Ip] is a command used to show/manipulate routing and devices.

Examples
--------

- **-h** forces human readable output:

    $ ip -h route
    
- **-d** gives a more detailed output:

    $ ip -d route

- **-s** gives more detailed time stats:

    $ ip -s route
